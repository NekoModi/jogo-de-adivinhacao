🎯 Jogo de Adivinhação

Um projeto simples, mas essencial para quem está começando na programação.

📌 Sobre o projeto

Este é um clássico jogo de adivinhação onde o usuário tenta descobrir um número aleatório dentro de um número limitado de tentativas.

Apesar da proposta simples, o objetivo principal é praticar conceitos fundamentais de desenvolvimento.

🚀 O que foi praticado
Variáveis e armazenamento de dados
Estruturas condicionais (if/else)
Repetições sobre eventos (loops)
Manipulação do DOM
Eventos no navegador (cliques e interações)
🧠 Aprendizados

Projetos como esse mostram que, mesmo algo aparentemente básico, é capaz de consolidar a base da programação.

Aqui, foi possível entender na prática como a lógica funciona e como o JavaScript interage com o HTML para criar experiências dinâmicas.

🛠️ Tecnologias utilizadas
HTML
CSS
JavaScript
🎮 Como funciona
O sistema gera um número aleatório
O usuário tenta adivinhar o número
O jogo informa se o número é maior ou menor
Há um limite de tentativas
O jogo pode ser reiniciado
